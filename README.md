# Живой вечер

Адаптивный лендинг творческого дуэта «Живой вечер» для презентации услуг, форматов мероприятий, отзывов, FAQ и формы заявки.

## О проекте

«Живой вечер» — это сайт творческого дуэта Елены Великановой и Ольги Рошко. Лендинг сделан в теплом, современном стиле и подходит для публикации на хостинге или загрузки на GitHub.

## Что есть на сайте

- первый экран с описанием дуэта
- блок о проекте
- раздел `Портфолио` с видео
- форматы мероприятий
- этапы работы
- стоимость
- отзывы
- FAQ
- форма заявки
- контакты

## Структура проекта

- `index.html` — основная HTML-страница
- `style.css` — стили сайта
- `prompt.md` — исходное техническое задание

## Как открыть локально

1. Откройте папку проекта.
2. Запустите файл `index.html` в браузере.

## Публикация на GitHub

Для загрузки проекта в GitHub обычно достаточно добавить в репозиторий:

- `index.html`
- `style.css`
- `README.md`

Если используются локальные изображения или другие медиафайлы, их тоже нужно загрузить в репозиторий рядом с основными файлами.
